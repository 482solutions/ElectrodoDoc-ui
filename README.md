## Overview
This UI part for Woden project.

## How to test:
1. Clone this repository
2. `yarn`
3. `yarn test`

## How to run:
1. Clone this repository
2. `yarn`
3. `yarn start`
4. Available at http://localhost:3000/

## How to deploy:
1. Clone this repository
2. `yarn`
3. `yarn build`
4. Copy `dist` directory to cloud storage
